TELEGRAM-STYLE SCROLLING IMPLEMENTATION
=====================================

This is a guide for implementing Telegram-like scrolling in React/Next.js apps.

STEP 1: CREATE THE CSS MODULE
----------------------------
Create a new file: styles/telegram-scroll.module.css

Copy this CSS:

.telegram-scroll {
  overflow-y: auto;
  scrollbar-width: thin;
  scrollbar-color: rgba(0, 0, 0, 0.2) transparent;
  overscroll-behavior: contain;
  -webkit-overscroll-behavior: contain;
  -webkit-overflow-scrolling: touch;
  scroll-behavior: smooth;
}

.telegram-scroll::-webkit-scrollbar {
  width: 6px;
}

.telegram-scroll::-webkit-scrollbar-track {
  background: transparent;
}

.telegram-scroll::-webkit-scrollbar-thumb {
  background-color: rgba(0, 0, 0, 0.2);
  border-radius: 3px;
  transition: background-color 0.2s;
}

.telegram-scroll::-webkit-scrollbar-thumb:hover {
  background-color: rgba(0, 0, 0, 0.3);
}

@media (prefers-color-scheme: dark) {
  .telegram-scroll {
    scrollbar-color: rgba(255, 255, 255, 0.2) transparent;
  }
  
  .telegram-scroll::-webkit-scrollbar-thumb {
    background-color: rgba(255, 255, 255, 0.2);
  }
  
  .telegram-scroll::-webkit-scrollbar-thumb:hover {
    background-color: rgba(255, 255, 255, 0.3);
  }
}

STEP 2: USE IN YOUR COMPONENT
----------------------------
Example usage:

import styles from '../styles/telegram-scroll.module.css'

const YourComponent = () => {
  return (
    <div className={styles.telegramScroll} style={{ height: '500px' }}>
      {/* Your content here */}
    </div>
  )
}

IMPORTANT NOTES
--------------
1. File must end with .module.css
2. Container needs a defined height
3. Use camelCase in component (styles.telegramScroll)

FEATURES
--------
- Thin scrollbar (6px)
- Smooth scrolling
- Dark mode support
- Disabled bounce effect
- Mobile-friendly
- Works in Chrome, Firefox, Safari, Edge

TROUBLESHOOTING
--------------
No scrollbar showing?
- Check if container has height set
- Verify content is taller than container

Bounce effect still present?
- Make sure overscroll-behavior properties are included
- Check -webkit-overflow-scrolling is set to touch

CSS not working?
- Confirm file ends with .module.css
- Check import path is correct
- Verify className syntax
